# 闪耀色彩角色卡面资源库

偶像大师闪耀色彩（THE IDOLM@STER SHINY COLORS）角色卡面图片资源库

## 📊 资源统计

- **总卡面数**: 818 张 WebP 格式
- **完整配对**: 408 组（基础版 + 觉醒版）
- **收录角色**: 32 位偶像
- **图片格式**: WebP（高质量压缩，quality: 85）
- **平均文件大小**: ~800 KB/张

## 📁 目录结构

```
闪耀色彩图片-最终版/
└── 角色卡面/
    ├── 主题名 角色名.webp          （基础版）
    └── 主题名 角色名+.webp         （觉醒版）
```

## 🎨 文件命名规则

- **基础版**: `主题名 角色名.webp`
- **觉醒版**: `主题名 角色名+.webp`
- **编码**: UTF-8（日文原名）

### 示例

```
優You 鈴木羽那.webp
優You 鈴木羽那+.webp

♡AKQJ10 有栖川夏葉.webp
♡AKQJ10 有栖川夏葉+.webp
```

## 🌐 CDN 使用

### GitHub Raw CDN

```
https://raw.githubusercontent.com/你的用户名/仓库名/main/角色卡面/主题名%20角色名.webp
```

浏览器会自动对URL中的日文和特殊字符进行编码。

### 在代码中引用

```typescript
const CDN_BASE = 'https://raw.githubusercontent.com/你的用户名/仓库名/main';
const imagePath = `${CDN_BASE}/角色卡面/${encodeURIComponent('優You 鈴木羽那.webp')}`;
```
